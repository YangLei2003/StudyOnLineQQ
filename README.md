# 智学堂 - 在线教学管理平台

一个基于 Spring Boot + Vue 的教学管理平台，涵盖教师管理、学生管理、作业管理、课程管理等核心功能，适用于中小型教育机构。

## ✨ 核心功能

### 教师端
| 功能 | 说明 |
| :--- | :--- |
| 课程管理 | 创建、编辑、发布课程 |
| 作业管理 | 布置作业、批改打分、发布成绩 |
| 学生管理 | 查看学生列表、学习进度 |
| 班级管理 | 创建班级、分配学生 |

### 学生端
| 功能 | 说明 |
| :--- | :--- |
| 课程学习 | 在线查看课程内容 |
| 作业提交 | 提交作业、查看批改结果 |
| 成绩查询 | 查看个人成绩和排名 |
| 个人中心 | 修改个人信息、密码 |

### 管理员端
| 功能 | 说明 |
| :--- | :--- |
| 教师管理 | 增删改查教师账号 |
| 学生管理 | 增删改查学生账号 |
| 数据统计 | 用户数量、活跃度统计 |

## 🛠️ 技术栈

| 模块 | 技术 |
| :--- | :--- |
| 后端框架 | Spring Boot 2.7 |
| ORM | MyBatis-Plus |
| 数据库 | MySQL 8.0 |
| 前端 | Vue 3, Element Plus, Vite |
| 状态管理 | Pinia / Vuex |
| 路由 | Vue Router |

### 环境要求
- JDK 1.8+
- MySQL 8.0+
- Maven 3.6+
- Node.js 16+
